Project Overview

This web application uses advanced machine learning algorithms to detect fraudulent credit card transactions in real-time. The system is trained on a highly imbalanced dataset and employs multiple techniques to handle class imbalance effectively.

Key Features:

  • Multiple ML algorithms comparison
  • SMOTE & Undersampling techniques
  • Real-time fraud prediction
  • Comprehensive evaluation metrics
  • Interactive visualizations
  • Feature importance analysis
  • ROC-AUC comparison
  • Confusion matrix analysis

Evaluation Metrics

Our models are evaluated using comprehensive metrics:

Accuracy

Overall correctness of predictions

Precision

Accuracy of fraud predictions

Recall

Ability to detect all frauds

F1-Score

Harmonic mean of precision & recall

ROC-AUC

Model's discrimination ability

Confusion Matrix

Detailed prediction breakdown

Important: In fraud detection, Recall is crucial as it measures our ability to catch all fraudulent transactions, even if it means some false alarms.
